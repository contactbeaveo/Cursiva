## 🆘 Besoin d'Aide ?

### Problème : Le build ne fonctionne pas
```bash
# Supprimez node_modules et recommencez
rm -rf node_modules package-lock.json
npm install
npm run build:verify
```

### Problème : Favicon ne s'affiche pas
1. Videz le cache : `Ctrl+Shift+R` (Windows) ou `Cmd+Shift+R` (Mac)
2. Attendez 24-48h (propagation DNS)
3. Vérifiez que `favicon.svg` est à la racine de votre site

### Problème : Sitemap donne 404
1. Vérifiez l'URL : `https://votresite.ca/sitemap.xml`
2. Assurez-vous que le fichier est à la racine (pas dans un sous-dossier)
3. Vérifiez les permissions : `644`

### Problème : Pages donnent 404
**Pour Apache :**
- Vérifiez que `.htaccess` est présent et actif

**Pour Netlify :**
- C'est déjà configuré dans `netlify.toml` ✅
